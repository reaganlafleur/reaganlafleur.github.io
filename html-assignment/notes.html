HTML Learning Notes

5 HTML Tags

Headings
Defines the hierarchy of text, where h1 is the highest level and h6 is the lowest level.
Paragraph
Defines blocks of text, essentially creating paragraphs with appropriate spacing and structure.
Anchor
Creates hyperlinks, which allows the user to navigate to other pages or other elements on a page.
Emphasis
Used to emphasize slighly more important text, in which the user will change their voice to read it and the audio software will add emphasis to the speech.
Image
Embeds images onto the webpage and also displays them using alt and src.
